# juno

👩‍🚀 Pretty logger for Node.

## installation

```sh
yarn add juno-js
```

## usage

```typescript
import { logger, globalOptions } from 'juno-js';

globalOptions.environment = 'production'; // optional, default is development

logger().info('Hi 🧕')
```
